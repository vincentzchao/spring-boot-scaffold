# spring-boot-scaffold

## 项目介绍

Spring Boot 项目脚手架

## 目录介绍

1. `base` 模块
    - 所有项目都必须依赖的基础模块。
2. `scenes` 目录
    - 存放一些具体的场景解决方案。
3. `starters` 目录
    - 存放一些模块化、开箱即用的模块。

## `scenes` 目录介绍

1. `user` 目录
    - 用户权限相关场景的解决方案。
